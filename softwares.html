Research softwares

  • El0ps : Python toolbox to solve L0-regularized problems via flexible and numerically efficient methods.
  • LIBSVMdata.jl : Simple tool to fetch LIBSVM datasets to Julia.
  • StochasticKEP.jl : Julia package to formulate Kidney Exchange Programs as stochastic optimization problems.

Contributed projects

  • BenchOpt : BenchOpt is a benchmarking suite for optimization algorithms. It is built for simplicity, transparency, and reproducibility.
  • Libsvmdata : A python util to fetch datasets from the LIBSVM website.
