Research softwares

  • El0ps.jl : Julia toolbox to solve L0-penalized problems using a sparsity-exploiting Branch-and-Bound algorithm.
  • BnbScreening.jl : A Branch-and-Bound solver for L0-penalized problems with Big-M constraint implementing Node-Screening tests.
  • StochasticKEP.jl : Julia package to formulate stochastic optimization problems linked to Kidney Exchange Programs. Elements of the problem data are not deterministic and represent the uncertainty about the compatibility between donors and patients. The decision variables in the formulations represent the matching choice and the recourse choice before and after the disclosure of this uncertainty.

Side and contributed projects

  • LIBSVMdata.jl : Simple tool to download and fetch LIBSVM datasets to Julia.
  • BenchOpt (contributed) : BenchOpt is a benchmarking suite for optimization algorithms. It is built for simplicity, transparency, and reproducibility.
  • Libsvmdata (contributed) : A python util to fetch datasets from the LIBSVM website.
