I have designed and coded a Branch-and-Price (BaP) algorithm to solve a groomer optimization problem in Julia during my internship at IRMAR with Jeremy Omer. Starting from no knowledge in Operation Research, I had to learn by myself this mathematical field. I managed to create and code a BaP algorithm to solve the problem in two months. Even if the algorithm is not fully usable in practice, I learned a lot from this project about how to tackle a mathematical problem.
