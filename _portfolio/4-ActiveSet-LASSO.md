I worked a bit on Active-set methods during my Turbo screening project and I found this method very interesting. I decided to try to code it and it seems to be very efficient to tackle the LASSO ! The code can be found in my Github repository.

For the theoretical part, take a look at Nocedal, J., & Wright, S. (2006). Numerical optimization. Springer Science & Business Media, it is a very good book.
